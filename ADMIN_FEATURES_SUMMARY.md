## 技术实现细节

### 前端框架
- Vue 3 (Composition API)
- Element Plus UI 组件库
- Element Plus Icons 图标库

### API 接口调用
- `/api/knowledge/list` - 获取知识列表（分页）
- `/api/knowledge` - 新增/更新知识
- `/api/knowledge/{id}` - 删除知识
- `/api/statistics/dashboard` - 获取统计数据
- `/api/statistics/hot-questions` - 获取热门问题
- `/api/consult/search` - 搜索对话记录
- `/api/statistics/recent-records` - 获取最近对话记录

### 状态管理
- 使用 Vue 3 的 `ref` 和 `reactive` 进行响应式数据管理
- 本地状态维护页面数据、分页信息、搜索条件等

### 错误处理
- try-catch 捕获异步操作异常
- Element Plus Message 组件显示错误信息
- 友好的用户提示

## 使用说明

### 启动应用
1. 确保后端 Spring Boot 应用已启动
2. 访问 `http://localhost:8080/admin.html`

### 主要操作流程
1. **查看数据**: 点击"数据看板"查看实时统计
2. **管理知识**: 在"知识库管理"中增删改查知识条目
3. **查看记录**: 在"对话记录"中查看用户咨询历史
4. **系统配置**: 在"系统设置"中配置基本参数

## 注意事项
1. 当前版本使用硬编码的租户 ID (tenantId=1)
2. 系统设置功能暂未连接后端接口
3. 退出登录功能仅做前端提示，需补充实际逻辑
4. 部分统计数据使用模拟数据，需对接真实接口
